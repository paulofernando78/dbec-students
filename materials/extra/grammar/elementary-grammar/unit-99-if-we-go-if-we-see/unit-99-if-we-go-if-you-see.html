Elementary Grammar

Unit 99 If we go ..., If you see ..., etc.

(First conditional)


A


"If" can be at the beginning of a sentence or in the middle:


"If" at the beginning.

If we go by bus, it will be cheaper mais barato.

If you don't hurry, you'll miss the train. Se (você) não se apressar, você ira perder o trem.

If you're hungry, have something to eat / grab something to eat.

If the phone rings, can you answer it, please?


"If" in the middle (without comma ',' sem vírgula)

It will be cheaper If we go by bus.

You'll miss the train if you don't hurry.

Is it OK if I use the phone?

I'm going to the concert if you get a ticket.


In conversation, we often use the if part of the sentence alone:

A: 'Are you going to the concert?'

B: 'Yes, If I can get a ticket.'


B


First conditonal: conditinal -> result

If you see Ann tomorrow ... etc.


After if, we use the present, not "will":

if you see Ann tomorrow, can you ask her to call me?

if I'm late this evening, don't wait for me. (not If I will be)

What shall we do if it rains? (not if it will rain)

If I don't feel well tomorrow, I'll stay at home.


C


If and when


Possibility (in the future)

If I go out = it is possible that I will go out, but I'm not sure mas não tenho certeza:


A: Are you going out later?

B: Maybe Talvez. If I finish my studies earlier, I'll call you.


For sure com certeza

When I go out = I'm going out

A: Are you going out later?

B: Yes, I am. When I leave work, I'll call you.


Compare when and if:

When I get home this evening, I'm going to have a shower.

If I'm late this evening, don't wait for me. (not When I'm late)

We're going to play tennis if it doesn't rain. (not when it doesn't rain)
